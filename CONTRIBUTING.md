# Contributing

Thank you for your interest in Anion!

## Repository Scope

This is a **documentation-only** public repository. The Anion source code is private during beta and security hardening.

## What You Can Contribute

Pull requests are welcome for:

- Typo fixes and grammar corrections in documentation
- Improvements to Mermaid diagrams
- Suggestions for clearer explanations
- Issue template improvements

## What Does Not Belong Here

- Source code (this repo has no application code)
- Binary files or large assets without prior discussion
- Content containing private information, API keys, or system paths

## How to Contribute

1. Fork this repository
2. Create a branch (`docs/your-change-description`)
3. Make your changes
4. Verify all markdown renders correctly and links are valid
5. Submit a pull request using the PR template

## Reporting Issues

Use the GitHub issue templates:

- [Bug Report](.github/ISSUE_TEMPLATE/bug_report.yml) — for issues with the Anion application
- [Feature Request](.github/ISSUE_TEMPLATE/feature_request.yml) — for feature suggestions
- [Beta Feedback](.github/ISSUE_TEMPLATE/beta_feedback.yml) — for general beta experience feedback
