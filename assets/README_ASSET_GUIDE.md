# Assets

This directory contains visual assets for the Anion public documentation repository.

## Structure

```
assets/
├── brand/          # Logo, icon, and brand assets
├── diagrams/       # Mermaid (.mmd) diagram source files
└── screenshots/    # Application screenshots
```

## Guidelines

All assets in this directory must be safe for public viewing. They must not contain:

- Private usernames or paths
- API keys or secrets
- Private logs or system information
- Internal repository references

See each subdirectory's README for specific guidelines.
