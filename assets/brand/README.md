# Brand Assets

This directory contains Anion's brand assets for the public repository.

## Available Assets

| Asset | Description |
|:---|:---|
| `anion_icon.png` | Anion application icon |

## Usage

- Use the icon in documentation, presentations, and social media
- Do not distort, recolor, or modify the icon
- Maintain adequate spacing around the icon

## Social Preview

For GitHub's social preview image (used when sharing the repo link):

- **Recommended size:** 1280×640px
- **Content:** Anion logo + tagline + key architecture diagram
- **Style:** Dark background, clean typography
