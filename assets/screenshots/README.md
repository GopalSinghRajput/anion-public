# Screenshots

This directory should contain clean, public-safe screenshots of Anion.

## Recommended Screenshots

| Filename | Description |
|:---|:---|
| `dashboard.png` | Main Anion desktop shell with Brain visualization |
| `aria-assistant.png` | ARIA assistant interaction panel |
| `terminal-intelligence.png` | Terminal risk preview and auto-recovery |
| `context-memory.png` | Context and memory dashboard view |
| `cross-device-sync.png` | Cross-device pairing and sync flow |

## Guidelines

Before adding screenshots, verify they do **not** contain:

- ❌ Private usernames or home directory paths
- ❌ API keys or tokens
- ❌ Private email addresses
- ❌ Private repository names
- ❌ Private logs or system secrets
- ❌ Local file paths that reveal user identity

If you need to redact information, use a solid-color block over the sensitive area.

## How to Add

1. Take a clean screenshot with no private information visible
2. Save as PNG at reasonable resolution (1920×1080 recommended)
3. Name the file according to the table above
4. Add to this directory
5. Verify the README references are correct
