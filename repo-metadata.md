## Recruiter Summary

Anion is a solo-developed, full-stack AI desktop system for Linux. It demonstrates:

- **Systems engineering:** systemd daemon architecture, FUSE filesystems, window manager IPC
- **AI engineering:** Bounded agentic AI (ReAct loop), tool allowlists, privacy-aware model routing
- **Full-stack development:** Python backend, React/Electron frontend, SQLite storage
- **Security awareness:** Automated SAST/SCA/DAST, guardrails, sandbox design
- **Testing discipline:** 742+ tests, chaos testing, 24-hour soak tests
- **Cryptography:** Custom NaCl Curve25519 E2E encrypted sync protocol
- **Product thinking:** Clear tradeoff documentation, honest beta positioning

## Short Technical Pitch

> Anion is a local AI operating layer for the Linux desktop that bridges the gap between your terminal, filesystem, and an agentic AI assistant — all running on-device with Ollama. It features ARIA (a bounded ReAct agent with 22 tools), terminal intelligence (risk preview + auto-recovery), cross-device E2E encrypted sync (NaCl/Curve25519), and a production-grade test harness (742+ tests, DAST, soak testing). Built as a solo project with Python, Electron/React, SQLite, and systemd.
